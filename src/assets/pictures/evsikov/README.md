# Инструкция по добавлению изображений

## Фото для страницы "Обо мне"

1. Загрузите ваше фото в эту директорию с одним из следующих имен:
   - `evsikov-photo.jpg` (рекомендуется)
   - `unnamed-6.jpg`
   - `photo.jpg`

2. **Важно:** Не используйте пробелы и скобки в именах файлов!

Фото будет автоматически подхвачено и отображено на странице "Обо мне".

---

## Дипломы и сертификаты

### Шаг 1: Загрузите изображения

Загрузите изображения дипломов и сертификатов в эту директорию (`src/assets/pictures/evsikov/`).

Поддерживаемые форматы:
- `.jpg` / `.jpeg`
- `.png`
- `.gif`

**Важно:** Используйте простые имена без пробелов и спецсимволов:
- ✅ `certificate-ozon.jpg`
- ✅ `diploma-bachelor.png`
- ❌ `диплом (копия).jpg` - НЕ ИСПОЛЬЗУЙТЕ кириллицу и скобки!

### Шаг 2: Добавьте имена файлов в код

Откройте файл `src/components/showcase/Certificates.tsx` и добавьте имена файлов в массив `CERTIFICATE_FILES`:

```typescript
const CERTIFICATE_FILES = [
    'certificate-ozon.jpg',
    'diploma-bachelor.jpg',
    'certificate-sber.png',
    // добавьте остальные файлы...
];
```
