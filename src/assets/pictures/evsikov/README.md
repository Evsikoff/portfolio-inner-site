# Инструкция по добавлению изображений

## Фото для страницы "Обо мне"

Загрузите ваше фото в эту директорию с одним из следующих имен:
- `evsikov-photo.jpg` (рекомендуется)
- `unnamed-6.jpg`
- `photo.jpg`

**Важно:** Не используйте пробелы и скобки в именах файлов, так как это вызывает ошибки при сборке.

## Дипломы и сертификаты

Загрузите изображения дипломов и сертификатов в эту директорию. Поддерживаемые форматы:
- `.jpg` / `.jpeg`
- `.png`
- `.gif`

Все изображения (кроме фото для страницы "Обо мне") будут автоматически отображены в галерее на странице "Дипломы и сертификаты".

## Пример структуры:

```
evsikov/
├── evsikov-photo.jpg          # Ваше фото для страницы "Обо мне"
├── certificate-1.jpg          # Сертификат 1
├── certificate-2.png          # Сертификат 2
├── diploma-bachelor.jpg       # Диплом бакалавра
└── ...                        # Другие сертификаты
```
